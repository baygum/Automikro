Tutorial Penggunaan Aplikasi

1

Masukkan koordinat lokasi pengukuran (Latitude dan Longitude) bisa klik pada peta atau tuliskan pada kolom koordinat.

2

Isi keterangan lokasi jika diperlukan (opsional).

3

Pilih mode upload: File Seismik (mseed, dll) atau Seismophile (bin + json), lalu Upload File Mikrotremor.

4

Klik tombol Olah Data untuk melihat hasil grafik HVSR dan interpretasi.
